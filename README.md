# LeetCode Problems

A personal collection of LeetCode solutions in JavaScript, built for learning and practice.

## Why This Repo?

This is my space to work through LeetCode problems at my own pace, track my progress, and strengthen my problem-solving and coding skills.

## Setup

```bash
npm install
```

## Scripts

| Command | Description |
|---|---|
| `npm run lint` | Check code for issues |
| `npm run lint:fix` | Auto-fix code issues |
| `npm run format` | Auto-format all files |
| `npm run format:check` | Check formatting |
